## Requirements:
- [x] People endpoint: Name, Hair color, Eye color, Birth year, Date created
- [x] Planets endpoint: Name, Climate, Population, Date created
- [x] Implement infinite scroll
- [x] simulator: iPhone XR
- [x] IDE Version: Xcode 10.2.1 
- [x] OS: iOS 12
